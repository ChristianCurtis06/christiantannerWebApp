## 🚀 Getting Started
 
### Prerequisites
 
- Node.js `v18+`
- npm or yarn
 
### Installation
 
```bash
# Clone the repository
git clone https://github.com/your-username/your-portfolio.git
 
# Navigate into the project
cd your-portfolio
 
# Install dependencies
npm install
 
# Start the development server
npm start
```
 
The app will run at `http://localhost:5173`.
